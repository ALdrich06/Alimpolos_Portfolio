CoopNexus Capstone Project

2nd Semester 2025-26

CoopNexus is a comprehensive cooperative management system designed to streamline operations for cooperatives. The platform features member management, loan processing, savings tracking, and dividend distribution with blockchain-integrated audit trail for transparent and immutable transaction recording. Built with modern technologies including React, Node.js, Express, PostgreSQL, and Ethereum blockchain integration for secure audit logging.

Key Features:

  • Member Management - Complete member registration, profile management, and membership tracking with blockchain audit trail
  • Loan System - Loan application, approval workflow, payment tracking, and interest calculation with immutable transaction records
  • Savings Management - Multiple savings types, deposit tracking, and withdrawal processing with blockchain verification
  • Dividend Distribution - Automated dividend calculation and distribution to members with transparent audit logging
  • Blockchain Audit Trail - Ethereum-based immutable logging of all critical transactions for transparency and accountability
  • Admin Dashboard - Comprehensive analytics, reports, and system management with audit trail visualization
  • Mobile App - React Native mobile application for members and cashiers with real-time transaction tracking
  • Multi-role Access - Different interfaces for Admin, Cashier, and Members with role-based permissions
