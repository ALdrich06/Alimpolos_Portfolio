# 🎓 Aldrich A. Alimpolos - Academic Portfolio

A modern, responsive portfolio website showcasing academic outputs and projects from the BSIT 3D program at the Institute of Computing, Mindanao State University.

![Responsive](https://img.shields.io/badge/Responsive-Yes-blue)

## 🌟 Features

- **Modern UI Design**: Glassmorphism effects, gradient backgrounds, and smooth animations
- **Fully Responsive**: Optimized for mobile, tablet, and desktop viewing
- **Subject Organization**: Projects organized by academic subjects
- **Interactive Elements**: Hover effects, transitions, and animated cards
- **Video Integration**: Embedded project demos and presentations
- **Downloadable Resources**: APK files, PDFs, and documentation

## 💻 Local Development

1. **Clone the repository**
```bash
git clone https://github.com/ALdrich06/Alimpolos_Portfolio.git
cd Alimpolos_Portfolio
```

2. **Open in browser**
```bash
# Simply open index.html in your browser
# Or use a local server:
python -m http.server 8000
# Then visit: http://localhost:8000
```

## 🌐 Deployment

### **GitHub Pages**

1. Create a new repository on GitHub
2. Push your code:
```bash
git remote add origin https://github.com/ALdrich06/[repo-name].git
git branch -M main
git push -u origin main
```
3. Go to Settings > Pages
4. Select branch: `main`, folder: `/ (root)`
5. Save and wait for deployment

Your site will be live at: `https://ALdrich06.github.io/[repo-name]`

### **Netlify** (Recommended)

1. Go to [Netlify](https://www.netlify.com/)
2. Drag and drop your project folder
3. Your site is live instantly!

## 📱 Responsive Design

The portfolio is fully responsive with breakpoints:
- **Mobile**: < 768px (1 column layout)
- **Tablet**: 768px - 1024px (2 column layout)
- **Desktop**: > 1024px (3-4 column layout)

## 🎨 Customization

### **Update Profile Information**
Edit `index.html`:
- Profile picture: `images/profile.jpg`
- Name, program, school info
- Stats cards (Subjects, Projects, Year)

### **Add New Projects**
1. Add project details in respective subject HTML file
2. Upload screenshots to `images/[subject]/`
3. Add videos to `videos/` folder
4. Update download links in `downloads/` folder

### **Change Color Scheme**
Main colors are defined in CSS:
- Primary: `#8b5cf6` (Purple)
- Secondary: `#ec4899` (Pink)
- Accent: `#06b6d4` (Cyan)
